# Motor_Controller

This repository contains electrical schematics of a motor controller designed for a Permanent Magnet Synchronous Motor (PMSM). The hardware platform enables implementation of Field-Oriented Control (FOC) algorithms and is intended for use in an integrated drivetrain system combining the motor, gearbox, and control electronics.
